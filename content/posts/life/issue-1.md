### 第一步-先保命
立即停车，开双闪，不要挪车，不要慌

保证人员安全（撤离到安全地带）

树立三角指示牌，城市道路50米，高速150米（警示牌在后备箱下部）

### 第二步-根据情况决定下面怎么做
有下列情况直接报警：有人受伤或死亡、双方对责任有争议

随后进行拍照取证，以下是必要的照片

额外记录一些信息：对方姓名、电话、身份证号、车牌号、车型、颜色、保险公司名称、保单号、事故发生事件和地点、目击者联系方式

### 第三步：责任认定及理赔
> 如果事故小，责任清晰，没有争议，则视频快处即可，交管12123中
>

其余则报警处理，交警到场后会：勘察现场，制作《交通事故责任认定书》，如对认定不服，收到认定书后三日内可向上一级公安机关申请复核

### 第四步：保险理赔
尽快找保险公司，平安的出险电话95511

然后保险公司会安排定损，会按照责任认定来决定；然后再修车，提交理赔材料，最后收到保险公司打款  
注意下面的：定损金额确认了再签字、不要私了后再报保险、修车有权选择去4S还是修理厂、保留所有发票和维修清单

不要认错、不要在取证完成前挪车、不要签空白文件、不要垫付医疗费、不要在定损完成前修车

#### 对方全责或主责时，可以主张的赔偿
车辆维修费、车辆贬值损失、交通费/代步费、拖车费救援费

医疗住院费、误工费（单位收入证明）、护理费、营养费、

#### 与网约车剐蹭的处理
首先确认对方是否三证齐全

| **<font style="color:rgb(255, 255, 255);">证件</font>** | **<font style="color:rgb(255, 255, 255);">说明</font>** |
| --- | --- |
| **<font style="color:rgb(0, 0, 0);">网络预约出租汽车运输证</font>** | <font style="color:rgb(0, 0, 0);">车辆的营运许可（贴在车上或电子版）</font> |
| **<font style="color:rgb(0, 0, 0);">网络预约出租汽车驾驶员证</font>** | <font style="color:rgb(0, 0, 0);">司机的从业资格</font> |
| **<font style="color:rgb(0, 0, 0);">平台经营许可证</font>** | <font style="color:rgb(0, 0, 0);">平台的资质（这个你不用操心）</font> |


如果对方三证齐全，这种情况下一定要报警，正常取证，额外拍三证、对面的所属平台、车上是否有乘客；

如果是我方主责或全责，对方有权利要求赔偿停运损失，但是需要提供收入证明（平台流水），以及合理的修车天数；停运损失一般是交强+商业三者承担

如果对方是非法营运，一定要报警，额外拍下对方的接单界面、车上乘客、对方无法出示证件的情况并且向交警说明。黑车有先天劣势，我们有额外谈判筹码，无法要求误工费，并且对方可能面临行政处罚

这里有一种额外的情况，对方全责、对方保险公司以非法营运拒赔，本人损失拿不到全额赔偿，可以：追究网约车平台责任、找保险公司走代位追偿

#### 何时应该请律师
对方全责拒赔，并且金额超过5000；涉及人身伤害、平台责任、保险公司定损不合理并且沟通无果、对方保险公司拒赔

#### 代位追偿
> 《中华人民共和国保险法》第六十条：
>
> "因第三者对保险标的的损害而造成保险事故的，保险人自向被保险人赔偿保险金之日起，在赔偿金额范围内代位行使被保险人对第三者请求赔偿的权利。"
>

代位追偿就类似于对方不赔，你的保险先赔你，然后保险公司再去找对方要钱

代位追偿流程

```java
第1步：事故发生，正常报警 + 取证
        │
第2步：拿到交警的《事故责任认定书》（确认对方全责/主责）
        │
第3步：联系对方的保险公司理赔 → 被拒绝/对方拖延/金额不够
        │
第4步：打电话给你自己的保险公司
        │
        "你好，我要申请代位求偿。
         事故责任认定对方全责/主责，
         但对方保险拒赔/对方个人无力赔偿/对方拖延不处理，
         我需要贵司先行赔付。"
        │
第5步：你的保险公司受理，安排定损
        │
第6步：提交材料（见下方清单）
        │
第7步：保险公司审核通过，赔款打到你账上
        │
第8步：你的保险公司拿着你给的材料，去找对方/对方保险公司追偿
        │
        （这一步你就不用管了，是保险公司自己的事）
```
